# **App Name**: Landmark Estates

## Core Features:

- Hero Section: Visually appealing hero section with company branding and introductory text.
- Featured Project Section: Showcase a featured real estate project with an image slideshow, project title, and brief description.
- Services Section: Highlight key services offered: Development, Property Management, and IT Solutions with a 'Learn More' button redirecting to a WhatsApp contact (+8801959162641).
- Projects Carousel: Implement an auto-sliding carousel showcasing various projects with an image, title, and 'View' button linking to a dedicated project details page.
- FAQ Section: A FAQ section with expandable dropdowns addressing common questions about Landmark New Homes Ltd.
- Contact Form with Admin Panel: A contact form that sends user messages to a Firestore database and can be viewed on a password protected admin dashboard.
- Admin Dashboard: Password-protected admin dashboard (/ad-panel, PIN: 5206) to manage projects, images, customer messages from the contact form; persists to a Firebase database.

## Style Guidelines:

- Primary color: Deep blue (#1E3A8A) to convey trust, stability, and professionalism.
- Background color: Light blue-gray (#F0F4F8) to provide a clean and modern backdrop.
- Accent color: Gold (#D4AF37) to highlight important sections and add a touch of luxury.
- Headline font: 'Playfair', a serif font for an elegant, fashionable, high-end feel.
- Body font: 'PT Sans', a san-serif font to complement 'Playfair' for longer text.
- Use simple, elegant line icons to represent different services and features.
- Maintain a clean, spacious layout with consistent margins and padding to improve readability and visual appeal.
- Subtle transitions and animations on hover states and when transitioning between sections to enhance user engagement.
